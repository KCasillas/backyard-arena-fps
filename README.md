# Backyard Arena FPS

Roblox Team Deathmatch prototype.

## MVP Goals

- 2 Teams (Red / Blue)
- 25 kills to win
- 5 minute match timer
- Respawn time: 2 seconds
- Weapons:
  - Rifle
  - Shotgun
- Hitscan damage
- Headshot multiplier
- UI:
  - Crosshair
  - Ammo counter
  - Team score
- 1 small arena map with cover
